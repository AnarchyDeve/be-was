💻 맥북 기본 단축키 정리
1. 시스템 및 파일 관리
* 복사: Command(⌘) + C
* 붙여넣기: Command(⌘) + V
* 파일 이동(잘라내기): Command(⌘) + Option(⌥) + V
* 실행 취소: Command(⌘) + Z
* 저장: Command(⌘) + S
* 앱 종료: Command(⌘) + Q / 창 닫기: Command(⌘) + W
* 앱 전환: Command(⌘) + Tab
2. 화면 캡처 및 도구
* 부분 화면 캡처: Command(⌘) + Shift + 4
* 전체 화면 캡처: Command(⌘) + Shift + 3
* 통합 캡처/녹화: Command(⌘) + Shift + 5
* 검색(Spotlight): Command(⌘) + Space bar
3. 웹 브라우저
* 새로고침: Command(⌘) + R
* 새 탭 열기: Command(⌘) + T
* 주소창 이동: Command(⌘) + L
4. 텍스트 및 터미널
* 이모지 입력: Command(⌘) + Control + Space bar
* 줄 맨 앞/뒤 이동: Command(⌘) + 방향키
* 터미널 강제 종료: Control + C
* 터미널 화면 청소: Command(⌘) + K
